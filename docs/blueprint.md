# **App Name**: Shree Label ERP

## Core Features:

- Estimate Module with Auto Layout & Costing: Calculate flexo label layouts (labels across, around, repeats, running meter) and costs (material, printing, UV, machine, labor, total cost per label, selling price, profit) with instant updates on the UI.
- Bill of Material (BOM) Management: Generate Bills of Material from estimates, detailing required materials, machine, die, plate, and ink, with an option to upload sample images.
- Sales Order Generation: Convert approved estimates into sales orders, automatically calculating the required raw materials based on the finalized order quantity.
- Inventory Tracking: Monitor and manage inventory for Jumbo Rolls, Slitted Rolls, Work In Progress (WIP), and Finished Goods, including barcode support for efficient tracking.
- Job Card Management: Create detailed job cards from estimates or sales orders, specifying machine, operator, materials, and production quantity for each job.
- Dashboard & Analytics: Provide an overview of key business metrics such as orders, production status, inventory levels, and sales performance through interactive charts.
- User Management & Authentication: Secure system access with robust user authentication and role-based authorization to manage access to different ERP modules and data.

## Style Guidelines:

- Primary color: Energetic yet professional orange (#E4892B) to signify vitality and clarity within the application. Chosen from HSL(25, 80%, 55%).
- Background color: A subtly warm, very light cream tone (#FAF6F2) for the main content area, providing a clean canvas. Visibly of the same hue as the primary color, chosen from HSL(25, 15%, 98%).
- Accent color: A deep, rich red (#A33131) for calls to action, alerts, and highlighting critical information, ensuring strong contrast. Analogous to the primary, chosen from HSL(0, 70%, 40%).
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable design, ensuring optimal clarity across data-intensive interfaces.
- Use a set of professional, monochrome, outline-based icons for clear and intuitive navigation and functionality across all modules, aligning with a corporate ERP aesthetic.
- Pezasys ERP-inspired layout featuring a fixed, dark left sidebar for consistent navigation and a primary content area organized with clearly defined cards and data panels for structured information display, ensuring a responsive and professional appearance.
- Subtle micro-animations will be implemented for data updates within forms and charts, seamless page transitions, and interactive element states, enhancing user feedback and the perceived responsiveness of the application.
